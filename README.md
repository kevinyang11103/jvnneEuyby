# 个人健康管理系统 java387

## 项目概述

个人健康管理系统是基于Springboot+Vue的前后端分离项目。该项目旨在为个人提供健康数据管理，方便用户更好地了解和关注自己的健康状况。

## 技术栈

- 后端：Spring Boot
- 前端：Vue.js

## 功能模块

- 用户管理
- 健康数据录入
- 数据可视化
- 健康报告生成

## 系统角色

- 普通用户：可以使用系统提供的健康数据管理功能
- 管理员：负责用户管理和系统维护

## 运行环境

- 后端：Java 1.8+，MySQL 5.7+
- 前端：现代浏览器

## 部署步骤

1. 准备后端运行环境：安装Java 1.8+和MySQL 5.7+
2. 导入数据库脚本
3. 配置后端应用属性
4. 部署后端服务
5. 前端使用npm或yarn安装依赖
6. 前端构建生产环境代码
7. 部署前端服务

## 目录结构

```
项目名称/
├── backend/               # 后端代码
├── frontend/              # 前端代码
```

## 核心亮点

- 前后端分离，便于模块化管理
- 数据可视化，直观展示用户健康状态
- 完善的用户管理功能，保障系统安全
